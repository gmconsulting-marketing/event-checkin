# Check-in Eventi

Strumento per il check-in dei prenotati agli eventi pop-up retail (Meet & Greet, Aperitivo Inclusivo).

Come si usa, dal tablet:
1. Apri la pagina pubblicata e aggiungila alla schermata Home.
2. Carica il file CSV o XLSX dei prenotati.
3. Cerca il nome e tocca "Presente" quando la persona arriva.
4. A fine giornata tocca "Esporta" per scaricare i risultati.

I dati dei prenotati restano solo sul dispositivo (localStorage): questa pagina pubblica contiene solo lo strumento vuoto.
